# 1MBD
One Managed Build &amp; Deployment

1MBD is a build framework which writen in Apache Ant. 
Feature as Below:
   :: it allows configured once,build everywhere no matter in local machine, ide, or build sever
   :: another featured is support Countinous Integration. 
   :: it very easy to extend to other project or product.
   :: and easy to linked to other build tools, these can become web ui for 1mbd. 
   :: not only that, easy write ur own UDF
   :: maven, u also can integrate with maven and typical build process
   :: it also can be act as build tool, as it support deploy and notification.
